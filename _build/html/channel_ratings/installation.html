Installation & Updates¶

Installing Channel Ratings¶

  1. Please ensure you are running at least ExpressionEngine 2.2 prior to installing the module.
  2. Copy channel_ratings module folder to your /system/expressionengine/third_party/ folder.
  3. Copy the channel_ratings themes folder to your /themes/third_party/ folder.
  4. In the ExpressionEngine control panel, head to Add-ons > Modules, and click ‘Install’ next to the module name.
  5. Make sure you set the Extension, Fieldtype and Module all to ‘Install’, and click Submit.

Channel Ratings is now installed. Please visit the docs for setup, usage, and templating

Updating Channel Ratings¶

  1. Backup your ExpressionEngine database and files on your web server!
  2. Did i already mention BACKUP!
  3. Strongly recommended: Test the upgrade process on a staging site first. If something goes wrong, you can always revert back!
  4. Replace the system/expressionengine/third_party/channel_ratings folder on your web server.
  5. Replace the themes/third_party/channel_ratings folder on your web server.
  6. In the ExpressionEngine control panel, head to Add-ons > Modules, and click on ‘Run Module Updates’ (in the top right)
